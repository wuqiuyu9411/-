一、var let const 的区别？ var 具有变量声明提升的作用 例： function person(status) { if(status) { var value = '123' } else { console.log(value) // undefined } } person(false) 可解析为：function person(status) { var value if(status) { value = '123' } else { console.log(value) // undefined } } 因此，var可以将变量提升到作用域最顶端，当var换成let 或const 将会报错 所以let const是块级作用域，var不是 在相同作用域中用let 和const 去声明一个已存在的变量，会报错 const可以修改对象的属性值，但是不能修改对象（不能修改指针） var声明的变量可能会挂在到window对象上。
